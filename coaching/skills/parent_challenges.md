# Skill: Navigating Parent-Specific Nutrition Challenges

**Purpose:** Equip coaches to address **recurring parent friction points** without shame — with **root cause → mindset reframe → practical strategy** for each pattern.

---

## How to Use This Skill

For each issue below: **validate → name the pattern → pick one strategy → set a 7-day experiment.** Never stack five fixes at once.

---

## 1. Eating Kids’ Leftovers (“Lick the Plate”)

| | |
|---|---|
| **Root cause** | Food waste guilt; mindless “completion” habit; mouth hunger vs stomach hunger; seeing food as “already paid for.” |
| **Mindset reframe** | “Finishing their plate doesn’t teach gratitude — it teaches **my needs don’t count.** I can respect food **without** being the human compost bin.” |
| **Practical strategy** | **Scrape-first rule:** Before you sit, scrape leftovers into compost/bin or labeled “tomorrow lunch” container. **Brush teeth** after kids’ dinner if picking is habitual. **Smaller kid portions** to reduce waste. **10-second pause:** “Am I hungry or tidying?” |

---

## 2. Late Dinners After Kids’ Bedtime

| | |
|---|---|
| **Root cause** | Chore compression; only “adult time” is paired with food/alcohol; underfueling daytime → ravenous night. |
| **Mindset reframe** | “Late eating isn’t ‘bad’ — **unplanned** late eating after starving all day **costs** sleep and morning choices. I’m allowed to **front-load** fuel.” |
| **Practical strategy** | **4–6 pm anchor snack** (protein + fiber + optional carb if training). **Split plate:** lighter at 9 pm if snack happened. **Slow-cooker / instant pot** so adult meal is ready when kids sleep. **Protein-first plate** even when late. |

---

## 3. Weekend Alcohol & Social Eating

| | |
|---|---|
| **Root cause** | Reward association after intensive parenting week; social lubricant; FOMO; liquid calories don’t register. |
| **Mindset reframe** | “I’m not ‘being good’ or ‘bad.’ I’m choosing **tradeoffs** that match **Monday me** and **training me**.” |
| **Practical strategy** | **Decide number + type before arrival** (e.g., 2 drinks, alternate water). **Eat protein before alcohol.** **Non-negotiable morning movement** Sunday if that supports identity. **Plan one meal** as flexible joy, **two meals** on structure. |

---

## 4. Stress Eating After Hard Parenting Days

| | |
|---|---|
| **Root cause** | Nervous system dysregulation; emotional exhaustion; food as fastest dopamine; restriction backfire. |
| **Mindset reframe** | “Food isn’t failing me — it’s **information**. My body wanted **relief**. I can add **relief tools** so food isn’t the only one.” |
| **Practical strategy** | **Non-food menu** on fridge (shower, 10-min walk, voice note vent, cold water face, playlist). **If-food plan:** portioned snack **plus** protein (e.g., apple + cheese) not “open bag on couch.” **HALT check** — Hungry, Angry, Lonely, Tired. |

---

## 5. Travel & Disrupted Routines

| | |
|---|---|
| **Root cause** | No kitchen; time zones; airport/grab-and-go defaults; “vacation mind” binary. |
| **Mindset reframe** | “**Maintenance** is a win on travel. I’m not chasing perfect — I’m **protecting protein and sleep**.” |
| **Practical strategy** | **Protein + produce at every airport meal** (even if imperfect). **Convenience buys:** ready boiled eggs, yogurt, jerky, pre-washed fruit. **Hotel kit:** shaker + protein powder, mixed nuts portion bags. **Rule of 2:** hit **2 of 3** — protein target, steps, sleep window. |

---

## Quick Coach Scripts (Universal)

- **Validation:** “That pattern makes sense given your schedule. We’re not fixing character — we’re **fixing the situation**.”  
- **One lever:** “If we only changed **one thing** this week, which would give the biggest relief?”  
- **Recovery:** “We’re **not restarting Monday**. Next **anchored** meal is your reset.”

---

## Example Output (Session Note)

**Issue:** Leftovers + late dinner.  
**Hypothesis:** Daytime underfueling.  
**Plan:** 3 pm Greek yogurt + fruit daily; scrape kids’ plates immediately; adult protein prepped in fridge for 8:45 pm assembly.  
**Check-in metric:** Evening hunger 1–10 + instances of picking.

---

## Red Flags (Refer Out)

Persistent binge-restrict cycles, purging, obsessive tracking, or training on dangerously low intake — coordinate with **physician / ED-specialist RD / therapist**.
