# Pacerai coaching skills

Read `.coaching_plans/<username>/profile.md` first whenever you work with a named athlete (see `AGENTS.md`).

The detailed protocols live as markdown under `coaching/skills/`. Open the file that matches the task:

| File | When to use |
|------|----------------|
| `client_intake_form.md` | New athlete onboarding; capture goals, schedule, health |
| `athlete_assessment.md` | Baseline fitness and readiness assessment |
| `training_plan_builder.md` | Building or revising multi-week `training_plan.md` |
| `weekly_checkin.md` | Full weekly check-in protocol |
| `weekly_checkin_template.md` | Structure/template for a check-in write-up |
| `pace_calculator.md` | Pace zones, conversions, workout pacing |
| `race_strategy.md` | Race pacing and execution |
| `injury_triage.md` | Pain/injury triage and escalation |
| `garmin_sleep_analysis.md` | Interpreting sleep data alongside training |
| `behavior_change_coaching.md` | Habits, adherence, motivation |
| `eating_systems.md` | Nutrition frameworks for active parents |
| `family_meal_prep.md` | Meal prep aligned to training |
| `parent_challenges.md` | Constraints for sports-active parents |
| `non_athlete_comms.md` | Messaging when the coachee is not the primary athlete |

Do not duplicate these documents in chat; follow the protocol in the relevant file and write outputs under `.coaching_plans/<username>/` as appropriate.
